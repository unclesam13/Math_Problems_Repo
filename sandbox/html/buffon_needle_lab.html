Monte Carlo Lab

Eksperyment igły Buffona

Ta symulacja pokazuje jednocześnie geometrię pojedynczego rzutu i zbieżność estymacji Monte Carlo. Najważniejszy warunek analityczny jest prosty: igła przecina linię wtedy i tylko wtedy, gdy odległość środka od najbliższej linii spełnia nierówność x ≤ (L/2) sin(θ).

Warunek trafienia: x ≤ (L/2) sin(θ)
Prawdopodobieństwo teoretyczne: p = 2L / (πd) dla L ≤ d
Estymata Monte Carlo: p̂n = liczba trafień / n

Jak czytać zmienne

  • x to odległość środka igły od najbliższej linii.
  • θ to kąt między igłą a liniami podłogi.
  • Przyjmujemy 0 ≤ x ≤ d/2 oraz 0 ≤ θ ≤ π/2.

Co pokazuje symulacja Monte Carlo

  • Każdy rzut to jeden punkt losowany w prostokącie parametrów (x, θ).
  • Zliczamy trafienia i liczymy estymatę p̂n.
  • Wykres pokazuje, jak estymata zbliża się do wartości teoretycznej.

Zbieżność estymaty i błąd

Masz dwa wykresy: estymatę Monte Carlo oraz moduł błędu względem wartości teoretycznej.

Gdy liczba rzutów rośnie, częstotliwość względna trafień zwykle stabilizuje się wokół wartości teoretycznej. Krótkie fluktuacje są naturalne i właśnie to widać na początku wykresu.
Ten wykres pokazuje błąd bezwzględny $| \hat p_n - p |$ po każdej kolejnej próbie. Dzięki temu łatwiej ocenić, czy większa próbka rzeczywiście poprawia jakość estymacji.
