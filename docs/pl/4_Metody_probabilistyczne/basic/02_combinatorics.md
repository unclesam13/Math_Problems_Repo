# Lista zadań 02 — Kombinatoryka dla prawdopodobieństwa

> **Uwaga:** Tłumaczenie w przygotowaniu. Ta strona jest placeholderem, aby zachować spójną numerację i te same nazwy plików jak w wersji EN.

## Co będzie na tej liście

Ta lista zadań będzie dotyczyć narzędzi kombinatorycznych używanych w rachunku prawdopodobieństwa, m.in.:

- zasady sumy i iloczynu,
- permutacje (w tym z powtórzeniami, kołowe),
- kombinacje,
- wariacje z/bez powtórzeń,
- rozróżnienie: kolejność / brak kolejności, powtórzenia / brak powtórzeń, obiekty rozróżnialne / nierozróżnialne.
